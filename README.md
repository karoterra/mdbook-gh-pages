# mdbook-gh-pages

mdBookで作ったページをGitHub Pagesでデプロイするのを試してみる。

ワークフローを実行する前に、リポジトリの「Settings -> Pages -> Build and deployment -> Source」を「GitHub Actions」にしておく。
